
通过官网命令创建 
选择ts， 简单版，（但是本项目移植了完全版的dashboard进来，移植进来的时候，根据run的错误往package.json里要加一些依赖，注意antcharts可能需要降级）


### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
